<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CI runner → build system | Gradle executes build scripts; malicious dependency could inject at build time |
| developer machine → Gradle Wrapper | Wrapper downloads binary from declared URL; hash must be pinned |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | gradle-wrapper.properties | mitigate | Pin distributionSha256Sum in gradle-wrapper.properties; Gradle 9 enforces checksum by default |
| T-01-02 | Tampering | libs.versions.toml | mitigate | Lock exact versions (not ranges) for all dependencies; no `+` or `latest` version specifiers |
| T-01-03 | Information Disclosure | .github/workflows/ci.yml | mitigate | Never echo secrets in workflow steps; use `${{ secrets.* }}` references only |
| T-01-04 | Elevation of Privilege | CI runner permissions | accept | Self-hosted runners are org-managed; no public repo access per CLAUDE.md constraint |
</threat_model>

```yaml
name: CI

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  build:
    runs-on: [arc-runner-unityinflow]

    steps:
      - uses: actions/checkout@v4

      - name: Set up JDK 21
        uses: actions/setup-java@v4
        with:
          java-version: '21'
          distribution: 'temurin'

      - name: Setup Gradle
        uses: gradle/actions/setup-gradle@v4

      - name: Lint (ktlint)
        run: ./gradlew lintKotlin

      - name: Build
        run: ./gradlew build

      - name: Test
        run: ./gradlew test

  arm64-build:
    runs-on: [orangepi]
    needs: build

    steps:
      - uses: actions/checkout@v4

      - name: Set up JDK 21
        uses: actions/setup-java@v4
        with:
          java-version: '21'
          distribution: 'temurin'

      - name: Setup Gradle
        uses: gradle/actions/setup-gradle@v4

      - name: Build
        run: ./gradlew build
```
