## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| filesystem → skill loader | Untrusted YAML files read from user-configured directory |
| skill content → agent system prompt | Skill prompt content injected into LLM system message |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | SkillLoader — filesystem YAML files | mitigate | Malformed YAML skipped with System.err warning (not crash). Jackson's YAMLMapper with `ALLOW_COMMENTS` disabled. Skills cannot reference external files or execute code — pure data class binding. |
| T-03-02 | Denial of Service | PatternMatcher — regex patterns | mitigate | Patterns are pre-compiled at load time.
